# BYD Apps

Custom Android apps for BYD electric vehicles.

Open-source apps for BYD vehicles running the DiLink infotainment system (Android-based head unit). These apps use BYD's internal Android APIs and are not available on Google Play.

## Compatibility

| Property | Value |
|----------|-------|
| Vehicle | BYD Dolphin 25/26 |
| Platform | DiLink 3.0 |
| Android | 10 (API 29) |
| Architecture | ARM64 (Qualcomm QCM6125) |

## Apps

| App | Description |
|-----|-------------|
| [Door Sound](apps/door-sound/) | Plays custom audio on door open/close/lock/unlock events |

## Prerequisites

A rooted head unit is required for sideloading apps. See [rooting guide](docs/) (coming soon).
